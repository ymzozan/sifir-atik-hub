# Sıfır Atık Hub — Yatırım Sunumu

Bu klasör, Sıfır Atık Vakfı'na sunulan yatırım destesinin kaynağıdır.

- **Çıktı:** `../Sifir-Atik-Hub-Yatirim-Sunumu.pdf` (13 slayt, A4 yatay)
- **Canlı demo:** https://sifir-atik-proje.vercel.app
- **Hedef kitle:** Sıfır Atık Vakfı yönetimi

## Akış (13 slayt)
Kapak → Vakıfla Uyum → Fırsat (problem/çözüm) → Nasıl Çalışır → Ürün →
Terimler (sade dille) → Paydaşlar → Veri Paneli → Etki → Fazlar →
Gelecek Vizyonu → Birlikte Ne Yapabiliriz → Kapanış

## Yaklaşım
- Doğrudan para talebi yok; "fırsat / iş birliği" dili.
- Teknik terimler (AI tanıma, oyunlaştırma, ESG) günlük dile çevrilir.
- Cesur fikirler (akıllı kutu, drone, robot) "uzun vadeli vizyon" olarak işaretlidir.
